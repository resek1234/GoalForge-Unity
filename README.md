## 📌 프로젝트 개요

**GoalForge Unity**는 Unity 2D로 제작된 빠른 템포의 1대1 축구 게임입니다.
두 플레이어가 같은 키보드를 사용해 대결하며, 슈퍼 모드를 활용한 전략적 플레이가 핵심입니다.

**핵심 컨셉**

- ⏱️ **1분 제한 시간** 동안 더 많은 골을 넣은 플레이어가 승리
- ⚡ **슈퍼 모드 게이지**: 자동 충전 + 공 터치 시 보너스
- 🎭 **캐릭터별 고유 슈퍼 모드**: 크기/속도 변화로 전략적 플레이 유도
- 🎮 **로컬 멀티플레이**: 한 화면에서 친구와 대결

## 🎮 게임 플레이

### 조작 방법

| Player                        | 이동            | 대쉬          |
| ----------------------------- | --------------- | ------------- |
| **Player 1** (왼쪽 파란 팀)   | `W` `A` `S` `D` | `Left Shift`  |
| **Player 2** (오른쪽 빨간 팀) | `↑` `←` `↓` `→` | `Right Shift` |

### 게임 규칙

1. **목표**: 1분 안에 상대 골대에 공을 더 많이 넣기
2. **슈퍼 게이지**
   - 시간이 지나면 자동으로 충전 (5배속)
   - 공에 닿을 때마다 게이지 10% 추가 충전
   - 게이지가 가득 차면 자동으로 슈퍼 모드 발동
3. **승리 조건**
   - 정규 시간 종료 시 점수가 높은 팀 승리
   - 동점일 경우 무승부 (DRAW)

## 💡 향후 추가 기능 아이디어

### 🎯 즉시 재미를 높일 수 있는 기능

#### 1. **파워업 아이템 시스템** ⭐⭐⭐

- 경기장에 랜덤하게 등장하는 아이템
  - ⚡ **스피드 부스트**: 5초간 이동속도 2배
  - �️ **실드**: 3초간 밀림 무효
  - ⚽ **메가 볼**: 다음 슛이 2배 강력
  - 🎲 **랜덤 박스**: 위 효과 중 하나 랜덤 획득
- **구현 난이도**: 중 / **재미 증가**: 높음

#### 2. **스킬 슛 시스템** ⭐⭐⭐

- 특정 키 조합으로 특수 슛 발동
  - 🌀 **커브 슛**: 공이 포물선으로 휘어짐
  - 💥 **파워 슛**: 직선으로 빠르게 날아감 (막기 어려움)
  - 🎯 **정밀 슛**: 조준점 표시 후 정확한 위치로 슛
- 쿨다운 또는 게이지 소모로 밸런스 조정
- **구현 난이도**: 중 / **재미 증가**: 높음

#### 3. **환경 장애물** ⭐⭐ ✅ 구현 완료

- 경기장에 랜덤 배치되는 장애물
  - 🧱 **벽**: 공과 플레이어를 막음
  - 🌪️ **바람 존**: 들어가면 한 방향으로 밀림
  - 🕳️ **함정**: 밟으면 1초간 이동 불가
- 매 경기마다 다른 배치로 리플레이 가치 증가
- **구현 난이도**: 중 / **재미 증가**: 중

#### 5. **콤보 시스템** ⭐⭐

- 연속으로 공을 터치하면 콤보 카운트 증가
  - 3콤보: 게이지 보너스 2배
  - 5콤보: 다음 슛 파워 증가
  - 10콤보: 즉시 슈퍼 모드 발동
- 화면에 콤보 카운터 표시로 긴장감 조성
- **구현 난이도**: 하 / **재미 증가**: 중

#### 6. **캐릭터 추가** ⭐⭐⭐

- 다양한 슈퍼 모드를 가진 캐릭터 추가
  - 🧊 **아이스맨**: 슈퍼 모드 시 필드에 얼음 생성 (미끄러짐)
  - � **파이어**: 슈퍼 모드 시 공이 불타며 더 빠르게 이동
  - 👻 **고스트**: 슈퍼 모드 시 3초간 투명화 + 벽 통과
- **구현 난이도**: 중~상 / **재미 증가**: 매우 높음

#### 7. **서든데스 모드 강화** ⭐

- 현재 동점 시 무승부 대신:
  - ⚡ **골든 골**: 먼저 넣는 팀이 즉시 승리
  - 🎯 **페널티킥**: 번갈아가며 슛 (골키퍼 AI 추가)
  - 💀 **데스 매치**: 골대 크기 점점 작아짐
- **구현 난이도**: 하~중 / **재미 증가**: 중

#### 8. **리플레이 & 하이라이트** ⭐

- 골 장면 자동 슬로우 모션 리플레이
- 경기 종료 후 베스트 플레이 모음
- **구현 난이도**: 중 / **재미 증가**: 중 (연출 강화)

#### 9. **커스터마이징** ⭐⭐

- 캐릭터 색상, 이펙트 변경
- 경기장 테마 선택 (잔디, 얼음, 우주 등)
- 공 스킨 (축구공, 농구공, 폭탄 등)
- **구현 난이도**: 중 / **재미 증가**: 중 (개성 표현)

#### 10. **토너먼트 모드** ⭐⭐

- 4강 → 결승 구조
- 각 라운드마다 다른 맵/규칙
- 우승 시 트로피 획득
- **구현 난이도**: 하 / **재미 증가**: 중 (목표 제공)
